UTILITY AND INSPECTION SCRIPTS
===============================

This folder contains utility scripts for inspecting, searching, and debugging scraped data.

Files:
------

1. search_province_codes.py
   - Utility script to search for province codes in scraped data
   - Helps identify province-specific requirements
   - Useful for debugging province mapping issues

2. inspect_alberta_html.py
   - Inspects Alberta-specific HTML structure
   - Helps understand HTML structure for Alberta requirements
   - Useful for debugging Alberta scraper issues

Usage:
------
- Run utility scripts: python utils/[script_name].py
- These are helper scripts for development and debugging
- Use when investigating specific data issues

Note:
-----
These scripts are primarily for development and debugging purposes.
