## ğŸ“‚ Key Directories Explained

### `src/` - Source Code
Main application source code:
- **components/**: Reusable UI components (Navigation, Footer, Modals, etc.)
- **pages/**: Page-level components (HomePage, PlannerPage, etc.)
- **context/**: React Context providers (AuthContext)
- **hooks/**: Custom React hooks (useDegreePlan)
- **data/**: Static JSON/JS data files
- **utils/**: Utility functions (Supabase client, storage helpers)

### `scraper/` - Data Scraping
Python scripts for scraping UBC data:
- Admission requirements
- Course information
- Prerequisites
- Province mappings

### `scripts/` - Build Scripts
Utility scripts for development and build:
- Copy scripts
- Data processing
- Course scraping helpers

### `Summary/` - Documentation
Organized documentation by category:
- **01-Project-Status/**: Current status and completion reports
- **02-Problem-Solutions/**: Problem-solving documentation
- **03-Guides/**: User and developer guides
- **04-Scraper-Documentation/**: Scraper-specific docs
- **05-Archives/**: Old/archived documentation

### `docs/` - Additional Docs
Planning documents and goals:
- Planning notes
- Future goals
- Development ideas

## ğŸ“„ Key Files

### Documentation Files
- **README.md**: Quick start guide and overview

### Configuration Files
- **package.json**: Node.js dependencies and scripts

### Data Files
- **src/data/**: All static data (JSON/JS files)

  - `firstYearRequirements.js`: First year course data
  - `curriculum/`: Engineering major curriculum files
  - `detailed_requirements_enhanced.json`: Province requirements
